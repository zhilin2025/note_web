v1.0 初始版本,采用indexeddb存储数据
v2.0 导航栏右侧新增了两个按钮：
-------------------------------------------------
📥 导出数据 — 点击后自动下载一个 notes_data_日期.json 文件到本地（浏览器默认下载位置），包含所有首页和项目数据。你把它移到 HTML 同目录即可。
📤 导入数据 — 选择之前导出的 JSON 文件，覆盖恢复全部数据到 IndexedDB 并重新渲染。
-------------------------------------------------
使用流程：
场景	操作
日常使用	正常编辑，自动保存到 IndexedDB
换浏览器打开	先在旧浏览器点「导出数据」，新浏览器打开 HTML 后点「导入数据」
重装系统前	把整个文件夹（含 HTML + JSON 文件）复制备份，重装后打开 HTML →「导入数据」恢复
定期备份	养成习惯定期点「导出数据」，JSON 文件和 HTML 放同一目录
首次打开且没有数据时，首页会显示黄色提示条提醒你可以导入。导入前会弹出确认，并建议先导出当前数据做备份。
